# Module 04: SPL Token-2022 (Token Extensions)

**Estimated Time:** 3-4 hours
**Prerequisites:** Module 03 (Anchor basics)

## Overview

Token-2022 is the modern token standard on Solana. It extends the original SPL Token Program with features like transfer fees, on-chain metadata, confidential transfers, and more. This module covers creating and managing fungible tokens with Token-2022.

## Learning Objectives

By the end of this module, you will be able to:

1. Explain the difference between SPL Token and Token-2022
2. Create a token mint with Token-2022 extensions
3. Configure transfer fees that auto-collect on every transfer
4. Add on-chain metadata (name, symbol, URI)
5. Mint tokens and manage supply
6. Revoke mint authority to create a truly fixed supply
7. Burn tokens programmatically
