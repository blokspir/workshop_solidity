# Module 06: Permanent Storage with Arweave / Irys

**Estimated Time:** 2 hours
**Prerequisites:** Module 05 (understanding of NFT metadata)

## Overview

NFTs store a URI that points to metadata and files. Those files need to be hosted permanently -- if the hosting goes down, the NFT becomes useless. Arweave provides permanent decentralized storage, and Irys is the SDK for uploading to it.

## Learning Objectives

By the end of this module, you will be able to:

1. Explain why permanent storage matters for NFTs
2. Compare storage options (IPFS, Arweave, centralized)
3. Upload files and JSON metadata to Arweave via Irys
4. Get permanent URIs for use in NFT minting
5. Estimate upload costs
