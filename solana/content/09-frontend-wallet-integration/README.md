# Module 09: Frontend Wallet Integration

**Estimated Time:** 4-5 hours
**Prerequisites:** Module 03 (Anchor), React/Next.js experience

## Overview

This module connects everything to a frontend. You'll learn how to build a Next.js app that connects to Solana wallets, reads on-chain data, and sends transactions.

## Learning Objectives

By the end of this module, you will be able to:

1. Set up `@solana/wallet-adapter` in a Next.js app
2. Connect Phantom, Backpack, and Solflare wallets
3. Send transactions from the browser
4. Read and deserialize on-chain account data
5. Build UI components for wallet connect, minting, and staking
6. Understand PWA basics for mobile wallet deep links
