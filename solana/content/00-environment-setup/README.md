# Module 00: Environment Setup

**Estimated Time:** 1-2 hours
**Prerequisites:** A computer with admin privileges, stable internet connection

## Overview

Before writing any Solana code, you need a working toolchain. This module walks you through installing every tool you'll need for the rest of the curriculum.

## Learning Objectives

By the end of this module, you will have:

1. Rust and Cargo installed via rustup
2. Solana CLI installed and configured for devnet
3. Anchor CLI installed and working
4. Node.js and Yarn set up for test scripts
5. VS Code configured with the right extensions
6. A devnet wallet funded with test SOL
7. Successfully deployed a "hello world" program to devnet
