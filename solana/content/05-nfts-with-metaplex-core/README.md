# Module 05: NFTs with Metaplex Core

**Estimated Time:** 3-4 hours
**Prerequisites:** Module 03 (Anchor), Module 04 (Token basics)

## Overview

Metaplex Core is the modern NFT standard on Solana. It replaces the older Token Metadata program with a simpler, more efficient design. This module covers creating collections, minting NFTs, managing metadata, and enforcing royalties.

## Learning Objectives

By the end of this module, you will be able to:

1. Explain NFT concepts on Solana
2. Compare Metaplex Core to legacy NFT standards
3. Create NFT collections
4. Mint NFTs with metadata and attributes
5. Set up royalty enforcement via plugins
6. Transfer and burn NFTs programmatically
7. Read NFT data from on-chain
