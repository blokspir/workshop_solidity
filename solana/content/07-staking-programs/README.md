# Module 07: Staking Programs

**Estimated Time:** 3-4 hours
**Prerequisites:** Modules 03-04 (Anchor, SPL tokens)

## Overview

Staking lets users lock tokens in a smart contract to earn rewards or unlock features. This module covers the program design, token vault patterns, reward calculation, and a complete staking implementation.

## Learning Objectives

By the end of this module, you will be able to:

1. Design a staking program with vault PDAs
2. Implement deposit and withdraw instructions
3. Calculate time-based rewards
4. Build a claim mechanism for accumulated rewards
5. Track per-user staking state
